TECHNOLOGIES AND ARCHITECTURE

Collaborative workspaces

Microsoft Teams
( Link )

Microsoft Teams is a software in which people are able to communicate to each other through channels, through video calling, voice call or even through text/chat.

We decided to use this tool since there is a built in record feature and it’s more secure. We are using the current version of teams for free.

Trello
( Link )

Trello is a free tool for collaboration and organization of projects and helps users to keep data on boards. It helps with users to tell who is working on what part of the project and what is in progress.

We use Trello to organize our project and share any important links that might be useful for our research. Trello has useful features such as checklists and labelling so each member knows what they’re working on, and figure out how long they have till the due date of each part of the project.

GitHub
( Link )

Github is a free tools tool that helps with the version control of our website’s development. This ensures that we are up-to-date with all our progress, and gives us a back-up of our previous versions in case we run into any issues through our development.

Our group decided to use Github because we were introduced to it in our intro to IT class so we are aware of its features, and we are able to use it without having any issues. Most developers also use github so it is easy to troubleshoot any problems that we might face.

AWS (Amazon Web Services)
( Link )

AWS(Amazon Web Services) is a secure cloud platform where the users can store databases and host websites. It also offers compute power.

We will use the latest AWS version because RMIT provided us with it for free and we also used it last semester so we know how to handle it.(write more later can't think anything else)

Google Docs
( Link )

Google Docs is a free tool that also helps the users organize their documents for a project and edit a document at the same time. They can produce many things like spreadsheets, surveys and slide presentations.

We decided to use google docs because we have used it in many assignments before and by now we know how to use it and its features, and also because it is very easy and very simple. We can work on something all together at the same time and make edits without affecting someone else’s work.

Google Sheets
( Link )

Google Sheets is a spreadsheet program, in which you can collaborate with other users in creating spreadsheets, through your computer or even your mobile phone.

We will use Google Sheets for our Gantt chart, to keep track of our weekly progress, and the percentage of the tasks required for the submission of the project.

Discord
( Link )

Discord is a free app that was firstly created for gamers and one person would host a server and people could join and all communicate at the same time. Nowadays discord is used not only for gaming but for other many reasons as well, for example creating a project with a team.

We use discord because as a team we think it is very easy to use and since everyone used it before hand we always have it opened and check messages/updates daily. Discord has a feature where you can add bots in servers and they can do multiple things. For example remind you for specific dates, record meetings, or even play music.

Software

Eclipse IDE
( Link)

Eclipse IDE is a free open-source software that would be useful in coding in Java. There is also an extension “marketplace” in which users may find useful plugins that may better their experience. It also has an error detection system in which the program would indicate visible errors, and stop the program from running.

We decided to use this software because most of us have either taken the classes Intro to Programming or Programming 1, meaning we have basic java knowledge. It is also easier to code mobile applications in Java rather than any other languages.

Tools

Visual Studio Code
( Link )

Visual Studio Code is a free source-code editor created by microsoft, with features such as a marketplace, embedded gits and autocompletion of a piece of code.

We decided to use Visual Studio Code because we all have used it before in previous subjects and we are experienced with its features. It is accessible by most operating system (Windows, Linux, and MacOS) which ensures that all of our members are able to access it without having to worry about the compatibility of their devices. The marketplace also gave us an advantage, especially with the online server and HTML preview, making it easier to edit our HTML files. We downloaded the latest version and the tool itself is free.

Figma
( Link )

Figma is a software/website focusing on vector graphics editing, and mostly for prototyping websites/apps due to it being able to setup interactivity for elements such as buttons to give life to the prototype.



Figma heavily influenced the efficiency of our diagrams and prototype, as it is easy to organize elements such as shapes and letters as you can name the elements to their corresponding type. Similarly, Figma allows us to collaborate within prototypes in real time, combined with MS Teams or Discord, we are able to collaborate in real time and figure out if there are any conflicts with the ideas of each member.
